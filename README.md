# nodeshelf

NodeShelf is a Free Blender Add-on (Currently under Development) that allows you to save/load node groups and use them across different blender files.
Create your personal Node Library with NodeShelf! and customize it with your own frequently used node groups!
Current version: 2.0
Demo Video (2.0 Update): https://www.youtube.com/watch?v=LJnNeExAj5w

What's New ?
-Node Packs: you can now categorize your node groups into packs depending on their use (math nodes, custom geometry, patterns.. etc) the possibilities are endless.
-Add/remove/rename Node Packs
-Assign Thumbnail images to your Packs
-Load selected Group from the active Node Pack
-Load the whole Pack at once
-Auto place loaded group

How to Use ?
-Create a new Node Pack
-Rename it and assign a thumbnail Image
-Select a Node Group
-Make sure your file has been saved AFTER that group has been created/last modified
-Click on the Save Group to Pack Button: you should see it being added to the drop down menu
-in a new blender file, choose your desired Node Pack & select a saved Node Group from the drop down
-Click on Load Group or Load Pack
-the group is now added to your blend file (if auto-place is enabled, the group will appear on the active node tree)
