# nodeshelf

NodeShelf is a Free Blender Add-on (Currently under Development) that allows you to save/load node groups and use them across different blender files.<br />
Create your personal Node Library with NodeShelf! and customize it with your own frequently used node groups!<br />
Current version: 2.0<br />
Demo Video (2.0 Update): https://www.youtube.com/watch?v=LJnNeExAj5w<br />

What's New ?<br />
-Node Packs: you can now categorize your node groups into packs depending on their use (math nodes, custom geometry, patterns.. etc) the possibilities are endless.<br />
-Add/remove/rename Node Packs<br />
-Assign Thumbnail images to your Packs<br />
-Load selected Group from the active Node Pack<br />
-Load the whole Pack at once<br />
-Auto place loaded group<br />

How to Use ?<br />
-Create a new Node Pack<br />
-Rename it and assign a thumbnail Image<br />
-Select a Node Group<br />
-Make sure your file has been saved AFTER that group has been created/last modified<br />
-Click on the Save Group to Pack Button: you should see it being added to the drop down menu<br />
-in a new blender file, choose your desired Node Pack & select a saved Node Group from the drop down<br />
-Click on Load Group or Load Pack<br />
-the group is now added to your blend file (if auto-place is enabled, the group will appear on the active node tree)<br />
